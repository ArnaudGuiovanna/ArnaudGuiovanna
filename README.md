## 🧪 What I'm Building

* **AI-native learning systems** — Not LMS with AI bolted on. Systems where the agent *is* the pedagogy engine.
* **Autonomous learning agents** — Agents that track cognitive state, adapt content sequences, and close the feedback loop without human intervention.
* **MCP-powered educational tools** — Turning AI models into persistent tutors with memory, planning, and tool use via the Model Context Protocol.

## 🧭 Philosophy

> "L'IA ne remplace pas l'enseignant. Elle rend l'apprentissage infini scalable."

I believe the next generation of learning tools won't look like apps — they'll look like agents. My work explores what happens when you design education systems natively around LLMs: no retrofitting, no wrappers. Just systems built from scratch with AI as the core reasoning layer.
